The Unofficial GitHub-Buttons
for MooTools / for jQuery

Show your Love to GitHub with the power of MooTools or jQuery
including buttons for stargazers, watchers, forks and followers

MooTools

Requires: Core/1.4.5, More/Number.Format, More/Request.JSONP

window.addEvent('domready', function(){
	// create a new stargazer button
	document.id('container1').grab(new GitHubButton({
		owner : 'AndiDittrich',
		repo : 'EnlighterJS',
		large : true,
		type : 'star',
		text : 'Starring:'
	}));
});

jQuery

Requires: jQuery/1.9

jQuery(function(jq){
	// create a new stargazer button
	jq('#container1').GitHubButton({
		owner : 'AndiDittrich',
		repo : 'EnlighterJS',
		large : true,
		type : 'star',
		text : 'Starring:'
	});
});
